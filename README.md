## 📌 Описание
Автоматизированный сбор расписания учебных занятий. Возможность транслировать расписание в MiniApp Telegram с помощью формата JSON, получаемого путем парсинга официального Excel-распсиания занятий учебного заведения** 

**Основные возможности:**
- Автоматическое скачивание актуального файла расписания с сервера по прямой ссылке
- Парсинг сложной структуры таблицы с использованием библиотеки Pandas
- Формирование JSON-файла для интеграции с внешними системами (ботами, сайтами)
- Валидация данных через автоматические тесты

## 🛠 Технологии
- **Python 3.10+**
- **Pandas & xlrd:** обработка табличных данных Excel старого формата
- **Requests:** взаимодействие с сетевым API сервера для загрузки файлов
- **Pytest:** автоматизированное тестирование функционала
- **Git:** контроль версий и ведение истории разработки

## 📂 Структура проекта
- `src/` — основной код (логика загрузки и парсинга)
- `data/` — локальное хранилище временных данных (в `.gitignore`)
- `tests/` — автоматизированные unit-тесты (`pytest`)
- `web/` — фронтенд часть для Mini App (развернута на GitHub Pages)
- `main.py` — точка входа в систему
- `requirements.txt` — список необходимых библиотек
